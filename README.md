# 운세 채팅 서비스

AI를 활용한 운세 상담 채팅 서비스입니다. Streamlit을 사용하여 구현되었습니다.

## 기능

- AI 기반 운세 상담
- 실시간 채팅 인터페이스
- 사용자 친화적인 UI
- 상세한 운세 분석 및 조언

## 설치 방법

1. 저장소 클론
```bash
git clone [repository-url]
cd [repository-name]
```

2. 필요한 패키지 설치
```bash
pip install -r requirements.txt
```

3. 환경 변수 설정
`.env` 파일을 생성하고 OpenAI API 키를 추가합니다:
```
OPENAI_API_KEY=your-api-key-here
```

## 실행 방법

```bash
streamlit run app.py
```
